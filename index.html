Aplicativos e sites

Não é nenhum segredo pra ninguém que existem milhares de sites e aplicativos para os mais diversos fins. Sabendo disso, por que não existiriam aplicativos e site para te ajudar a programar, fazer seu próprio site já não é mais uma tarefa da qual você necessita de um computador. Então, aqui vai uma lista e alguns motivos pelos quais voce deveria tentar fazer isso pelo seu celular. Afinal, quando se trata de aprender alguma coisa, atualmente, só não aprende quem não quer.

    Codepen

Esse é um site ótimo para programar com html, css e javascript. Por ser um site, caso tenha algum laptop ou computador é facilmente acessível por ficar salvo na nuvem do site. Nele também é possivel exportar o seu trabalho para onde quiser, o site ainda inclui um salvamento automatico (é triste perder um projeto porque não salvou ;-; ) e um leyout bem intuitivo.

    anWriter

anWriter é um aplicativo simples, mas bem feito, esse aplicativo tem um tipo de salvamento automatico que funciona, mas é sempre bom ter certeza salvando seus arquivos manualmente, todos os aquivos são salvos localmente. O aplicativo está disponivel para android apenas, e utiliza do próprio sistema android para testar os codigos. Nele é possivel programar em html, css, javascript e LaTeX. Além disso o aplicativo contém um sistema de auto completar.

    TrebEdit

TrebEdit é outro aplicativo ótimo, o que pode ser um problema nele é que alguma funções são pagas. Pode ser um problema se você não tiver dinheiro ou não souber que pauzinhos mexer. Esse aplicativo tem um layout simples e limpo, pode ser usado para estudar programação e nele é possivel até ver os codigos de sites, basta colocar o url e pronto.

É claro que existem outros aplicativos e sites que podem ser usados, mas pessoalmente eu acho que se você quiser começar esse é um ótimo ponto de partida. Se quiser ter aulas para porgramação, aplicativos como Mimo podem ser um ponta pé inicial muito bom, ou caso inglês não seja o seu forte, existem canais no youtube que ensinam maravilhas.
